# Autonomous_Cars


👉 [مشاهده README فارسی](READMEpersian.md)



This repository offers a Persian adaptation of the "Autonomous Cars" course, tailored for Persian students to better grasp concepts like lane detection, object recognition, and deep learning in selfDriving vehicles. It aids classroom comprehension. Though primarily a localization, select sections are updated with recent articles and modern methods.
